Plugins

Os plugins extendem as funcionalidades do WordPress e permitem fazer o que você quiser!

Hooks

“Hooks são fornecidos pelo WordPress para permitir que o seu plugin (ou alguma funcionalidade do tema) adicione ações (actions) ou filtros (filters) específicos em ganchos, ou seja, para ativar funções em pontos específicos da execução do WordPress.”

Actions

São disparadas quando algo acontece no WordPress.
Por exemplo:

  • Quando o site carrega
  • Quando um post é publicado
  • Quando um usuário é criado

Filters

Também são disparados quando algo acontece no WordPress, porém manipulam as informações.
Por exemplo:

  • Modifica as informações antes de salvar no banco ou exibir na página
  • Modifica a forma como as informações devem ser exibidas
